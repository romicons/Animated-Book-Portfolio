PROYECTOS

Te invito a visitar algunos de mis proyectos, desarrollados con mucha dedicación y atención a los detalles.

PORTFOLIO

Enfatizando en la estética y el diseño, este atractivo portfolio implementa animaciones de menú atractivas con jQuery y JavaScript, y prioriza la validación básica de formularios para asegurar tanto la accesibilidad como una experiencia de usuario fluida.

JUEGO DE MEMORIA

Desarrollado principalmente con JavaScript, este cautivador juego incorpora características dinámicas, diseño adaptable y una interfaz de usuario fluida, demostrando mi capacidad para crear experiencias interactivas y agradables para los usuarios.

POKEDEX

Construido con React, este proyecto fue diseñado para explorar los fundamentos de esta tecnología. Aprovechando los hooks esenciales como useState, useEffect y useContext, diseñé una interfaz de usuario dinámica que interactúa de manera fluida con PokeAPI.

GENERADOR DE MEMES

Esta aplicación interactiva permite a los usuarios añadir imágenes, superponer textos y editar dinámicamente ambos elementos utilizando diversas funcionalidades de JavaScript. Se centra en crear una experiencia web atractiva y fácil de utilizar mediante implementaciones innovadoras de JavaScript.
